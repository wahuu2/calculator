# 🧮 Calculator

A simple, elegant, and responsive **Calculator application** built with **HTML, CSS, and JavaScript**.  
This app allows users to perform basic arithmetic operations with a clean UI, smooth interactions, and a toggleable dark mode.

---

## ✨ Features
- ➕ Addition, ➖ Subtraction, ✖️ Multiplication, ➗ Division
- 🧹 Clear all input (AC)
- ❌ Delete last entry (DEL)
- 🟰 Compute results instantly
- 🌗 Light/Dark mode toggle
- 📱 Responsive design for mobile and desktop

---

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/your-username/calculator-app.git
cd calculator-app
```

### 2. Open in browser
Simply open `index.html` in your browser — no server setup required.
